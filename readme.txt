The instructions for compilation are as follows below:

If you want to compile it, open a terminal emulator and type in the below
command:

gcc C_Example.c

It will compile a output file called a.out.

If you want to run this a.out file, run the command below:

./a.out

It should output the below text:

This is a C Program, and it's my first one too.
In case you wanted to know, C is a pretty simple language to learn.

If you don't have gcc installed, type this into your terminal below:

Debian/Ubuntu/Linux Mint:
sudo apt-get install gcc

Fedora/Yellow Dog/Red Hat Enterprise/CentOS:
yum install gcc

FreeBSD:
pkg install gcc
